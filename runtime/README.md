# Runtime Dependencies

## Camoufox Browser
Download from: https://github.com/daijro/camoufox
Place `camoufox` folder here.

## Embedding Model
Download `sentence-transformers/all-MiniLM-L6-v2` from HuggingFace:
https://huggingface.co/sentence-transformers/all-MiniLM-L6-v2

Or let the system auto-download on first run.
